<h3 id="q&a">三、常见问题解答：</h3>

<h3 id="is-nethersx2-better">1、NetherSX2 是否比 AetherSX2 运行得更快？</h3>

<p>视情况而定。<br/>
主要侧重点是稳定性和游戏兼容性，而非性能，但一些 GameDB 的更改可能对某些游戏的性能有益，也可能有害。<br/>
如果存在不影响稳定性和兼容性的性能修复，通常在发现后会被加入。<br/>
因此，请尝试一下，看看是否适合您。</p>

<h3 id="minimum-requirements">2、有什么要求？</h3>

<p>PS2 主机是一个极其复杂的硬件，拥有许多非常强大的组件，即使在今天看来也是如此。由于主机硬件与您手机硬件的差异，模拟器在计算上非常耗费资源。您需要高端设备才能获得良好的性能。</p>
<p>我们建议至少使用相当于骁龙845的设备。这意味着需要4个大核（Cortex-A75 级别）。如果您只有两个大核（例如：骁龙700系列SoC），则不应启用多线程VU，性能会因此受到影响。配备Mali 或 PowerVR GPU的设备可以运行此应用，但性能将低于Adreno GPU。</p>
<p>如果您想在较慢的设备上使用此应用，可以尝试，但游戏运行会较慢，尤其是负载较重的游戏。</p>

<h3 id="the-app-tells-me-i-need-a-bios">3、应用提示我需要BIOS。</h3>

<p>是的，您需要。BIOS镜像是玩游戏<strong>所必需的</strong>，不是可选的。此镜像应使用自制软件从您自己的主机上提取。网上有很多关于如何提取主机BIOS的指南。</p>

<h3 id="where-do-i-get-games">4、在哪里获取游戏？</h3>

<p>NetherSX2 应仅用于玩您拥有并已自行从光盘提取的游戏。要提取游戏，您可以使用 ImgBurn 等工具创建光盘的 .iso文件，然后通过USB将其复制到您的设备上。您也可以选择使用 gzip 或 chdman 等压缩工具来进一步减小文件大小。</p>

<h3 id="best-settings">5、某某游戏的最佳设置是什么？</h3>

<p>默认设置通常对大多数游戏效果最佳。<br/>
使用Vulkan渲染器并将硬件下载模式设置为"禁用回读"可以获得额外的性能提升。</p>

<h3 id="my-games-are-running-slowlagging">6、我的游戏运行缓慢/卡顿。</h3>
<p>不同的游戏对硬件的要求差异很大，这取决于它们对主机各组件的利用程度。请参阅下一节。</p>

<h3 id="how-to-improve-performance">7、如何提高性能？</h3>
<ul>
    <li>使用Vulkan渲染器，<strong>特别是如果您有 Mali GPU</strong>。请注意，某些游戏使用OpenGL可能性能更好。</li>
    <li>对于某些Mali设备，在高级设置中启用<strong>线程化呈现</strong>可以在非EE瓶颈时显著提高性能（例如：Pixel 6）。</li>
    <li>在系统设置中，通过将循环速率设置为负数，循环跳过设置为正数来降频模拟的CPU。这会破坏某些游戏，并导致其他游戏运行速度低于预期。</li>
    <li>如果游戏速度减慢取决于摄像机角度，这可能是由于GS下载造成的，这在移动GPU上非常慢。您可以尝试将硬件下载模式设置为"不同步"或"禁用"，但这可能会导致一些特效出现故障。</li>
    <li>您可以尝试将SPU2混合模式切换到"异步混合"而不是"时间拉伸"。但这并非与所有游戏兼容。</li>
    <li>对于某些设备（例如三星和Pixel），使用关联性控制可以提高性能一致性。请参阅下面的关联性控制部分。</li>
    <li>降低混合精度可以改善某些游戏的性能，但代价是画面可能出现错误。</li>
    <li><strong>如果在更新后性能变差，请重置设置。</strong> 有时我们会将设置更改为性能更好的默认值。</li>
</ul>

<h3 id="onscreen-buttons">8、如何添加 L3/R3 按键和右摇杆？</h3>
<p>在游戏中暂停并点击右上角的控制器图标打开控制器设置。选择切换按键以添加 L3/R3，和/或将控制布局更改为双摇杆。</p>
<h3 id="how-do-i-customize-the-touchscreen-controller-positionscale">9、如何自定义触摸屏控制器的位置/大小？</h3>
<p>在游戏中按暂停或返回按键，然后点击右上角的控制选项卡。您还可以在此处添加快捷键的附加按键，例如快进、快速加载/保存等。</p>

<h3 id="the-app-opens-in-portrait-mode-how-do-i-change-it-to-landscape">10、应用以竖屏模式打开，如何更改为横屏？</h3>
<p>如果您启用了自动旋转，请旋转您的设备。您也可以在应用设置的第一页中强制其始终使用横屏模式。</p>

<h3 id="my-bluetooth-controller-isnt-working">11、我的蓝牙控制器无法使用。</h3>
<p>在"控制器设置"中映射控制器。您可以从自动映射开始，但有时扳机键或摇杆仍需要手动映射。</p>

<h3 id="i-want-to-set-different-settings-for-each-game">12、我想为每个游戏设置不同的配置。</h3>
<p>在游戏列表/网格中长按游戏，点击"游戏属性"，然后点击"游戏设置"选项卡。如果您想在游戏内更改这些设置，请打开暂停菜单，然后点击右上角的信息按键以访问"游戏属性"。</p>

<h3 id="input-profiles">13、我想为不同的游戏使用不同的触摸屏按键布局。</h3>
<p>NetherSX2 现在拥有"输入配置文件"，它是对触摸屏按键布局以及蓝牙控制器绑定的分组。</p>
<p>创建输入配置文件的方法：</p>
<ol>
  <li>按照您想要的布局配置触摸屏按键（可选）。</li>
  <li>用您想要的配置绑定蓝牙控制器，并设置任何宏按键（可选）。</li>
  <li>将其保存为输入配置文件。在控制器设置中，第一个选项卡有一个"保存配置文件"选项。</li>
  <li>（可选）将触摸屏布局重置为默认值（也在同一菜单中）。</li>
</ol>

<p>您可以在需要时手动加载此配置文件，但也可以为特定游戏自动加载配置文件。操作方法如下：</p>
<ol>
  <li>为您想要使用配置文件的游戏打开"游戏属性"（可以在游戏内或从游戏列表打开）。</li>
  <li>滑动到"常规"选项卡。</li>
  <li>点击"输入配置文件"，然后选择您希望使用的配置文件。</li>
  <li>当游戏启动时，配置文件将自动加载。当切换到另一个游戏时，布局将恢复为全局设置。</li>
</ol>

<h3 id="macro-buttons">14、我想创建宏或"组合键"。</h3>
<p>每个虚拟控制器有4个宏按键。
<ol>
  <li>打开控制器设置</li>
  <li>滑动到您想为其创建宏的控制器端口（通常是端口 1）。</li>
  <li>向下滚动到"宏"类别，点击您想自定义的四个按键之一。</li>
  <li>（可选）为宏按键设置一个触发器。这可以是您手机上的一个按键（例如音量键）、控制器按键、控制器按键组合等。或者，如果您想通过触摸屏激活它，请留空。</li>
  <li>选择您希望宏激活的按键组合。您可以选择任何功能按键、摇杆方向等的组合。例如，X键 + 右摇杆向上。</li>
  <li>频率设置决定了宏将以多少帧的间隔进行切换。如果您只想要一个组合键，请将频率保持为零/默认。否则，它将按照您选择的频率模拟重复松开/按下按键。换句话说，它自动化了连按按键！</li>
  <li>点击弹出窗口外部，或按返回键退出宏设置。</li>
  <li>（可选）启用宏的触摸屏按键。它们默认是隐藏的。这可以在"控制器设置" -> 切换按键中找到，然后选择宏1到宏4。您必须在游戏中才能添加按键。按键将显示为 M1 到 M4。</li>
</ol>

<h3 id="my-games-have-rendering-glitches">15、我的游戏有渲染故障。</h3>
<p>由于主机硬件的复杂性，在使用硬件渲染器时仍然会出现大量问题。您可以尝试对这些游戏使用软件渲染器。</p>

<h3 id="i-want-to-save-more-than-one-state">16、我想保存多个即时存档。</h3>
<p>打开暂停菜单并点击"加载/保存存档"，有10个槽位 + 一个快速存档（用于屏幕快速存读档按键）。</p>

<h3 id="i-want-copy-my-saves-from-another-device">17、我想从另一台设备复制我的存档。</h3>
<p>目前您只能一次性导入整个记忆卡；无法导入单个存档。在游戏列表中从左侧滑动将显示"导入记忆卡"选项，您可以使用它来导入记忆卡的 *.ps2 镜像文件。</p>

<h3 id="where-are-my-saves-located">18、我的存档位于哪里？</h3>
<p>由于 安卓11+ 上的分区存储限制，我们无法将您的存档放在外部存储的正常目录中。但是，使用文件管理器应用，您应该能够访问主存储卷中的 <code class="language-plaintext highlighter-rouge">Android/data/xyz.aethersx2.android</code> 目录，该目录包含您的即时存档和记忆卡。请注意，在 安卓11+ 上访问此目录需要向您的文件管理器授予额外权限。</p>

<h3 id="how-do-i-add-covers-to-the-game-grid">19、如何为游戏网格添加封面？</h3>
<p>将封面图片以 jpg/png 格式，使用游戏标题或编号作为文件名，放在上述数据目录下的 covers 目录中。或者，您可以在游戏列表中长按并选择"选择封面图片"来导入封面。</p>

<h3 id="how-do-i-create-launcher-shortcuts-for-games">20、如何为游戏创建启动器快捷方式？</h3>
<p>在游戏列表中长按游戏，然后选择"创建启动器快捷方式"。</p>

<h3 id="software-renderer-performance">21、如何提高软件渲染器的性能？</h3>
<p>软件渲染器使用CPU而不是GPU来绘制屏幕上的一切，相比之下速度非常慢。然而，软件渲染器的性能现已改进到足以用于许多游戏的程度。</p>
<p>假设您有一台拥有4个大核的设备，您需要<strong>禁用多线程VU</strong>以释放一个大核，并将软件渲染线程数设置为2。然后您可以启用关联性控制（任何模式），这也会将软件线程固定到大核上。</p>
<p>如果您的设备只有2个大核，软件渲染器的性能永远不会好；小核对光栅化来说太慢了。</p>

<h3 id="affinity-control">22、如何使用关联性控制？</h3>
<p>对于某些设备（例如三星和Pixel），使用关联性控制可以提高性能一致性。启用此选项后，您设备中的CPU核心会按从最强到最弱的顺序排序，线程将按照您选择的顺序固定到这些核心上，以避免调度程序不断移动它们。</p>
<ol>
  <li>EE线程（情感引擎）- 此线程负责大部分游戏逻辑，以及VU0、DMA等。在大多数游戏中，它是最繁重的线程。</li>
  <li>GS线程（图形合成器）- 此线程负责接收来自EE和VU1的数据包，对其进行分析，并为宿主GPU生成渲染所需的绘制命令。在某些游戏中，尤其是那些绘制调用繁重的游戏，它可能是最繁重的。</li>
  <li>VU线程（矢量单元1）- 此线程处理VU1微程序的执行，其功能类似于现代GPU的顶点着色器。它通常是最轻的线程，但在某些游戏中（例如《战神》），由于复杂的微程序和/或同步，它实际上可能是最繁重的。此线程仅在启用MTVU时处于活动状态。</li>
</ol>
<p>最佳设置取决于您的设备和游戏，因此它是可按游戏设置的。如果您遇到巨大的帧率波动，波动范围超过20fps，尝试调整此选项可能会有所帮助。</p>
<p>使用图形设置中的"显示CPU使用率"选项来确定哪个线程是最繁重/第二繁重的，然后选择相应的关联性控制模式。具有对称大小核分布的设备可能不需要使用此选项。</p>
